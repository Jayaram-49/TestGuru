# TestGuru - Software Requirements Specification (SRS)

## 1. Introduction

### 1.1 Purpose

The purpose of this Software Requirements Specification (SRS) document is to define the functional and non-functional requirements of the TestGuru Online Examination Platform. It serves as a reference for planning, designing, developing, testing, and maintaining the application throughout its development lifecycle.

### 1.2 Scope

TestGuru is a web-based online examination platform designed for educational institutions, instructors, and students. The platform allows instructors to create and manage examinations, students to take exams online, and administrators to manage users and monitor the system. The project aims to provide a secure, user-friendly, and efficient examination experience.

### 1.3 Intended Audience

This document is intended for:

- Developers
- Project Maintainers
- Testers
- Future Contributors
- Anyone who wants to understand the project requirements
