# TestGuru

## Overview

TestGuru is a modern web-based Online Examination Platform designed for educational institutions and learners.

The platform enables instructors to create and manage examinations, students to take exams online, and administrators to manage the overall system efficiently.

This project is being developed as a learning journey to understand real-world software engineering practices using modern technologies.
